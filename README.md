# Calcolo timbratura

Versione PWA pronta per GitHub Pages.

## File principali

- `index.html`
- `manifest.webmanifest`
- `service-worker.js`
- `icons/`
- `.nojekyll`

## Deploy su GitHub Pages

Carica tutti i file nella root del repository, poi abilita GitHub Pages da **Settings > Pages** selezionando il branch principale e la cartella root.

Le icone PWA usano sfondo blu Polimi e simbolo orologio bianco, senza scritte.
